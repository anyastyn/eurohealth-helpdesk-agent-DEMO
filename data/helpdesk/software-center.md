# EuroHealth Software Request and Installation

Last reviewed: 2026-02-18
Owner: End User Computing

## Approved installation path

Employees should install standard software from `Software Center` when available.

## Standard flow

1. Open `Software Center`.
2. Search for the required application.
3. Select `Install`.
4. Wait for the installation status to show `Installed`.
5. Restart the device if the application requires it.

## If the app is missing

- Submit a ServiceNow request under `Software / New Application`
- Include business justification, urgency, and manager name
- Security review may be required for browser extensions, AI tools, and file-transfer utilities

## Support rule

The helpdesk can explain the request process, but cannot approve unlisted software directly in chat.
