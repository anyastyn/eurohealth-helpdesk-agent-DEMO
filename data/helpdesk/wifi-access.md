# EuroHealth Office Wi-Fi Access

Last reviewed: 2026-01-30
Owner: Workplace Services

## Approved office networks

- `EH-Corp` for managed employee devices
- `EH-Guest` for temporary guest access

## Employee device connection

1. Open the Wi-Fi settings on the managed laptop.
2. Select `EH-Corp`.
3. Sign in with the EuroHealth email address if prompted.
4. Accept the corporate certificate only if it is issued by the approved EuroHealth trust chain.
5. Wait for the device to show `Connected, secured`.

## Common issues

- If `EH-Corp` is missing, toggle Wi-Fi off and on and move closer to the access point.
- If credentials are rejected after a password change, wait up to 15 minutes for sync.
- If the device loops on certificate prompts, open an incident for Workplace Services.

## Guest network rules

`EH-Guest` must not be used for internal admin tools or systems with employee data.
